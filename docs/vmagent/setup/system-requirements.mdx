- Odigos Pro license key (or token) for the VM Agent.
- A Linux host. For the [package installation](/vmagent/setup/installation): a RHEL-compatible distribution (DNF),
a Debian-based distribution (APT), or any Linux system supporting RPM packages. For the
[Docker deployment](/vmagent/setup/docker): Docker installed and running, with permission to start privileged containers.
- At least one process or application on the host that is written in a 
[supported language](/enterprise/instrumentations/overview).

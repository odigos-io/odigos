## `JsonAttributeCondition` <a id="JsonAttributeCondition"></a>
    

**Appears in:**

- [AttributeCondition](#AttributeCondition)

<p>JsonAttributeCondition defines conditions for evaluating JSON-encoded span attributes.
It supports structural and value-based filtering using key lookups and JSONPath expressions.</p>
<p>Supported operations:</p>
<ul>
<li>&quot;exists&quot;: Checks that the attribute is a non-empty string.</li>
<li>&quot;is_valid_json&quot;: Ensures the string parses as valid JSON.</li>
<li>&quot;is_invalid_json&quot;: Ensures the string is not valid JSON.</li>
<li>&quot;equals&quot;: Compares the full JSON string to expected_value.</li>
<li>&quot;not_equals&quot;: Verifies the full JSON string is not equal to expected_value.</li>
<li>&quot;jsonpath_exists&quot;: Asserts that the specified json_path resolves to a non-empty result.</li>
<li>&quot;key_equals&quot;: Compares the value at json_path to expected_value.</li>
<li>&quot;key_not_equals&quot;: Verifies the value at json_path is not equal to expected_value.</li>
</ul>

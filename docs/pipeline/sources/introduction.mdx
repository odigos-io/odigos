A Source in Odigos is any component that can generate opentelemetry signals - traces, metrics and logs.

## Supported Sources

Odigos supports the following sources:

- Kubernetes Deployment
- Kubernetes DaemonSet
- Kubernetes StatefulSet

Which are refered to as kubernetes workloads.

The following programming languages / runtimes are supported:

- golang
- python
- nodejs
- java
- dotnet

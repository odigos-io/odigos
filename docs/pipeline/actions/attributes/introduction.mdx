## Attributes Action Types

- [Add Cluster Info](/pipeline/actions/attributes/addclusterinfo) - Add static cluster-scoped attributes to your data.
- [Delete Attribute](/pipeline/actions/attributes/deleteattribute) - Delete attributes by name from your data.
- [Rename Attribute](/pipeline/actions/attributes/rename-attribute) - Rename attributes by name in your data.
- [Pii Masking](/pipeline/actions/attributes/piimasking) - Mask attributes value by PII Category.

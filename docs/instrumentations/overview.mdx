Instrumentations in Odigos is the code that is used to record observability signals from your services.
Odigos will automatically apply instrumentation to your selected workloads to collect traces, metrics, and logs without any code changes.

## Supported Runtimes

Odigos provides automatic instrumentations in kubernetes for the following runtimes:

- [Go](/instrumentations/golang/golang) versions >= 1.17
- Java versions >= 17
- [Node.js](/instrumentations/nodejs/nodejs) versions >= 14
- [Python](/instrumentations/python/python) versions >= 3.8
- .NET

## odigos sources enable rollout

enable a Rollout for Odigos instrumentation

### Synopsis

This command enables the provided Rollout for Odigos instrumentatin. It will create a Source object if one does not already exists, or update the existing one if it does.

```
odigos sources enable rollout [name] [flags]
```

### Options

```
      --dry-run            dry run
  -h, --help               help for rollout
  -n, --namespace string   Kubernetes Namespace for Source (default "default")
```

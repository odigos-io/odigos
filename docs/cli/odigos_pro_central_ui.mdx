## odigos pro central ui

Port-forward Odigos Central UI and Backend to localhost

### Synopsis

Port-forward the Central UI (port 3000) and Central Backend (port 8081) to localhost to enable local access to Odigos UI.

```
odigos pro central ui [flags]
```

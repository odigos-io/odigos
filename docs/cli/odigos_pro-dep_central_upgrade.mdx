## odigos pro-dep central upgrade

Upgrade Odigos Central UI in the central namespace

```
odigos pro-dep central upgrade [flags]
```

### Options

```
      --central-max-message-size string   Maximum message size in bytes for gRPC messages (empty = use default)
  -h, --help                              help for upgrade
      --image-pull-secrets strings        Secret names for imagePullSecrets (repeat or comma-separated)
  -n, --namespace string                  Target namespace for Odigos Central upgrade (default "odigos-central")
      --version string                    Specify version to upgrade to
      --yes                               Confirm the upgrade without prompting
```

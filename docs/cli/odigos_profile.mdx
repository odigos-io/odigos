## odigos profile

This command can be used to list Odigos effective profiles

### Synopsis

```
odigos profile
```

### Examples

```

# List Odigos effective profiles
odigos profile

# Display the profiles available for the relevant tier
odigos profile --available
```

### Options

```
  -a, --available   list all available profiles to use
  -h, --help        help for profile
```

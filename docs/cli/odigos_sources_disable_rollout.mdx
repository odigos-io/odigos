## odigos sources disable argo-rollout

disable an Argo Rollout for Odigos instrumentation

### Synopsis

This command disables the provided Argo Rollout for Odigos instrumentation. It will create a Source object if one does not already exists, or update the existing one if it does.

```
odigos sources disable argo-rollout [name] [flags]
```

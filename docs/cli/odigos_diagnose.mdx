Retrieves Logs of all Odigos components in the odigos-system namespace and CRDs of Actions, instrumentation resources and more.<br />
The results will be saved in a compressed file for further troubleshooting.

## Synopsis

This command can be used for troubleshooting and observing Odigos state.
The file will be saved in this format: odigos_debug_ddmmyyyyhhmmss.tar.gz


```bash
odigos diagnose
```

## Output

```
odigos_debug_25092024102322.tar.gz
```
